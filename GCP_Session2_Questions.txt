Google App Engine

1. In App Engine - Standard, the pricing is based on,
	(a) Number of CPUs
	(b) Instance Hours
	(c) Used memory
	(d) Number of user requests

	Ans: b
	
2.	App Engine deployment configurations will be available in,
	(a)package.json
	(b)pom.xml
	(c)app.yaml
	(d)requirements.tx
	
	Ans: c

3. In App Engine - Flexible, we can scale down to 0 instances if there are no requests from the user.
	(a) True
	(b) False

	Ans: b
